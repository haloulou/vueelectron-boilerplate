# vueelectron-boilerplate
Projet de base Electron + Vue + Vite + Vuetify + Pinia + Vue Router

Un projet de départ pour créer des applications desktop avec Electron, Vite, Vuetify, Vue Router et Pinia. Ce template est conçu pour gagner du temps et éviter de repartir de zéro à chaque nouveau projet.

Technologies incluses:
- Electron: pour créer des applications Desktop multiplateformes.
- Vite: Build ultra-rapide pour Vue.
- Vuetify3: pour la création des interfaces utilisateurs.
- TypeScript: Langage typé de JavaScript permettant de détecter des erreurs avant l'exécution du code.
- Vue Router: Navigation entre les vues.
- Pinia: Gestion d'états.

Installation:
bash
# Cloner le projet
git clone https://github.com/haloulou/vueelectron-boilerplate.git
cd vueelectron-boilerplate

# Installer les dépendances
npm install

# Exécuter l'application en mode développement
npm run dev
